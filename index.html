¿Quién soy?

Hola, soy Enzo Moreyra, Téc. Desarrollador de Software y este es mi portfolio, un espacio web para subir mi CV, proyectos, asimismo ,compartir conocimientos, curiosidades y pasión por aprender acerca de este mundo IT.

Los lenguajes de programación que manejo son C++, C#, Python, Javascript, Php, HTML5, CSS, Bootstrap, React, SQL, MongoDB.
Actualmente, trabajo en un proyecto de Sistema pensado para punto de ventas de un comercio pequeño o mediano. El programa permite gestionar las ventas, el flujo de caja, gestionar el stock de productos, precios y los clientes con sus respectivas cuentas corrientes.
El programa es de escritorio, para montarlo en máquinas con escasos recursos. El lenguaje usado es C#.

También estoy desarrollando proyectos web y tengo que decidir por un proyecto para el trabajo final de la carrera en Programación III. Por lo que estaré confirmando el proyecto pronto.
